# CS2

## Enable Terminal

To enable the terminal in CS2, follow these steps:
1. Open CS2.
2. Go to the "Settings" menu.
3. Go to "Game".
4. When you see "Enable Developer Console", set it to "Yes".

## Important Commands

Press `~` to open the console. Commands you can use:
```
mp_maxrounds 5     // Set max rounds to 60
mp_warmup_end      // End warmup
mp_restartgame 1   // Restart the game after 1 second
```

## Increase Mouse Sensitivity

1. Open CS2.
2. Go to the "Settings" menu.
3. Go to "Keyboard/Mouse".
4. Under "Keyboard and mouse settings" look for "Mouse sensitivity". 
5. Adjust the slider to increase or decrease mouse sensitivity as needed.

## Ajust Audio Settings

1. Open CS2.
2. Go to the "Settings" menu.
3. Go to "Audio".
4. Adjust the "Master Volume" and "Main Menu Ambience Volume".
